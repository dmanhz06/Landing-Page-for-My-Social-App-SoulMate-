Mạng xã hội di động thế hệ mới SoulMate

Kết hợp độc đáo giữa việc kết nối cộng đồng, không gian nghe nhạc thư giãn và góc viết nhật ký chia sẻ cảm xúc cá nhân bằng giọng nói thời gian thực.

Android Platform Java / Kotlin Firebase Backend

🚀 Tính Năng Nổi Bật

Hệ thống giải pháp thông minh tối ưu hóa trải nghiệm tương tác

Xác Thực An Toàn

Đăng ký, đăng nhập bảo mật qua Firebase Authentication. Tích hợp nhanh chóng bằng Google, Facebook.

Tương Tác Thời Gian Thực

Cập nhật bảng tin (feed), thả cảm xúc, bình luận và chat tức thời bằng sức mạnh Firebase Realtime / Firestore.

Social Music & Diary

Không gian cá nhân lý tưởng giúp bạn vừa thư giãn nghe nhạc vừa ghi lại những mảnh ghép nhật ký cảm xúc hàng ngày.

Ghi Âm Nhật Ký (Audio)

Tự động nhận diện giọng nói và chuyển đổi hội thoại thành văn bản hoặc lưu trữ trạng thái âm thanh trực tiếp.

Giao Diện Dark Mode

Hệ thống Theme Management thông minh, linh hoạt chuyển đổi Light/Dark Mode giúp bảo vệ mắt và tiết kiệm pin.

Thông Báo Đẩy Tức Thời

Tích hợp giải pháp OneSignal gửi thông báo đẩy nhanh chóng về các tương tác, tin nhắn mới, duy trì kết nối.

🛠️ Công Nghệ Sử Dụng

Kiến trúc sạch mẫu mực đảm bảo tính bảo trì lâu dài

Frontend Stack

Java / Kotlin

Xây dựng ứng dụng Native chuyên nghiệp trên nền tảng Android SDK cốt lõi.

Backend & Database

Google Firebase

Vận hành mượt mà qua Authentication, Realtime Database, Cloud Functions và Firestore.

Cloud Storage & Media

Cloudinary & OneSignal

Quản lý và nén tối ưu file truyền thông đa phương tiện chất lượng cùng thông báo đẩy.

Architecture Pattern

MVVM Pattern

Tách biệt rõ ràng cấu trúc luồng dữ liệu đem lại mã nguồn sạch, mở rộng tốt.
